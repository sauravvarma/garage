# Brief & direction

## Purpose
Streakly is a habit & streak tracker: a single screen where a user sees each habit, its current streak, and can mark it done for today.

## Audience
Individuals building daily habits on their phone. Mobile-first, glance-and-tap usage.

## Tone
Calm, focused, lightly motivating. The streak is the hero; chrome stays out of the way.

## Success criteria
A user can see their streaks and complete a habit in under two taps, on a phone, without confusion about state (done / not done / gated).

## Constraints
SvelteKit + TypeScript, mobile-only, dark theme only. Free tier capped at 3 active habits.
